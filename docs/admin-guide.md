# Skylark 管理后台使用指南

> 管理后台仅对企业 Owner（管理员）开放，入口位于侧栏或工作台「管理后台」快捷入口。
>
> 所有管理操作均会写入审计日志，可在「操作日志」页面查看。

- 总成员数
- 总会话数
- 总消息数

## 成员管理

**路径**：`/admin/members`

### 成员列表

| 功能 | 说明 |
|------|------|
| 搜索 | 支持按姓名、邮箱、工号、手机号搜索 |
| 部门筛选 | 下拉选择部门，仅展示该部门成员 |
| 列表字段 | 头像、姓名/邮箱、部门、职位、工号、角色、入职时间 |

### 成员详情编辑

点击成员行可打开右侧详情面板，可编辑以下信息：

| 字段 | 说明 |
|------|------|
| 姓名 | 修改用户显示名 |
| 性别 | 男 / 女 / 未知 |
| 手机号 | 联系电话 |
| 部门 | 下拉选择（数据来自部门管理） |
| 职位 | 岗位名称 |
| 工号 | 企业内部编号 |
| 工作城市 | 办公所在城市 |
| 角色 | `owner`（管理员）或 `member`（普通成员） |

底部提供「移除成员」操作（需二次确认）。

### 邀请成员

点击「邀请成员」按钮打开邀请弹窗：

1. **邮箱邀请**：输入一个或多个邮箱（逗号或换行分隔），设置有效期（1天/3天/7天/30天/永不过期），点击发送
2. **邀请码**：页面顶部展示企业邀请码，可一键复制分享

系统自动校验：
- 已是企业成员的邮箱将跳过
- 已有未过期邀请的邮箱将跳过

### 邀请记录

切换至「邀请记录」标签页查看：

| 字段 | 说明 |
|------|------|
| 邮箱 | 被邀请人邮箱 |
| 状态 | 待接受 / 已接受 / 已过期 / 已取消 |
| 邀请人 | 发起邀请的管理员 |
| 创建时间 | 邀请发出时间 |
| 过期时间 | 邀请链接失效时间 |

操作：
- **复制链接**：复制邀请链接发送给被邀请人
- **取消邀请**：取消未使用的邀请（状态变为「已取消」）

### 邀请链接流程

1. 管理员在后台创建邮箱邀请
2. 被邀请人打开链接 `/invite/{id}`，看到企业信息和邀请详情
3. 已登录用户直接点击「接受邀请」加入企业
4. 未登录用户先跳转登录/注册，再返回接受邀请

## 部门管理

**路径**：`/admin/departments`

### 组织架构

- 以**树状结构**展示企业部门层级
- 每个部门节点显示名称和成员数

### 部门操作

| 操作 | 说明 |
|------|------|
| 新建部门 | 输入名称，可选择父部门 |
| 编辑部门 | 修改部门名称、调整父部门 |
| 删除部门 | 删除后成员的部门字段清空（不影响成员本身） |

## 机器人管理

**路径**：`/admin/bots`

### 创建机器人

填写以下信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | 是 | 机器人显示名 |
| 描述 | 否 | 机器人功能说明 |
| 头像 URL | 否 | 自定义头像 |
| Webhook URL | 否 | 接收消息回调的地址 |
| Webhook Secret | 否 | 用于验证回调来源 |

创建后自动生成唯一 API Token（格式：`sk-bot-xxxxx`）。

### 机器人列表

展示所有机器人（含已停用），显示：名称、状态、描述、API Token（可复制）、Webhook 配置、创建者、创建时间、订阅会话数。

### 机器人操作

| 操作 | 说明 |
|------|------|
| 编辑 | 修改名称、描述、头像、Webhook 配置 |
| 启用/停用 | 切换机器人状态，停用后 API 调用将返回 403 |
| 重新生成 Token | 旧 Token 立即失效，生成新 Token |
| 删除 | 彻底删除机器人及其所有订阅 |

### 开发对接

机器人创建后，开发者可使用 API Token 调用 Bot API 实现自动化功能，详见 [Bot API 文档](./bot-api.md)。

## 会议室管理

**路径**：`/admin/rooms`

### 添加会议室

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | 是 | 如"鲲鹏会议室" |
| 楼栋 | 否 | 如"A栋" |
| 楼层 | 否 | 如"3F" |
| 房间号 | 否 | 如"301" |
| 容量 | 否 | 可容纳人数 |
| 设施 | 否 | 多选：投影仪 / 白板 / 视频设备 / 电话 |
| 状态 | 是 | 可用 / 维护中 / 停用 |

### 会议室列表

- 按楼栋分组展示
- 搜索会议室（名称/楼栋/楼层）
- 每个卡片展示：名称、位置、容量、设施标签、状态

### 会议室操作

| 操作 | 说明 |
|------|------|
| 编辑 | 修改所有属性 |
| 删除 | 删除后已关联的日程中会议室字段自动清空 |

### 预订逻辑

- 会议室预订在**日历-创建日程**时进行
- 选择时间段后系统自动查询可用会议室
- 存在时间冲突的会议室将提示「时间冲突」并禁止选择
- 日程创建成功后会议室自动关联

## 加入申请审批

**路径**：`/admin/join-requests`

- 展示所有待处理的企业加入申请
- 申请人信息：姓名、邮箱、申请时间
- 操作：批准（加入企业为 member）或拒绝

## 操作日志

**路径**：`/admin/logs`

记录管理员的所有操作行为，审计追踪。

| 字段 | 说明 |
|------|------|
| 操作类型 | 如：创建成员、编辑部门、删除机器人、创建会议室 等 |
| 操作对象 | 被操作的实体名称/ID |
| 详细信息 | 具体变更内容 |
| 操作人 | 执行操作的管理员 |
| 时间 | 操作发生时间 |
