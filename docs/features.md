## 即时消息（IM）

### 单聊

- 从通讯录发起与联系人的 1v1 私聊
- 自动复用已有会话（get-or-create）
- 展示对方姓名和头像

### 群聊

- 在消息页面一键创建群聊
- 搜索并勾选组织内成员加入群组
- 自定义群名称
- 群聊中展示成员名称标识

### 机器人对话

- 在工作台点击机器人「对话」按钮发起与机器人的 1v1 会话
- 用户消息自动触发 Webhook 推送，机器人可通过 API 实时回复
- 机器人会话名称以 🤖 前缀标识

### 实时通信

- 基于 Cloudflare Durable Objects 的 WebSocket 实时消息推送
- 消息发送采用**乐观更新**策略，体验流畅
- 支持断线自动重连（3 秒间隔）
- 在线/离线状态广播
- "正在输入"状态提示
- 已读状态同步

### 消息功能

- 文本消息
- 图片消息（上传至 R2 + 预览）
- 文件消息（上传至 R2 + 下载）
- 时间分割线（超过 5 分钟间隔自动插入）
- 消息气泡区分自己/对方，带头像和昵称

### 会话列表

- 按最后消息时间倒序排列
- 智能时间展示（今天显示时分、昨天、本周几、月/日）
- 支持搜索过滤会话
- 切换会话时**仅右侧内容区刷新**，会话列表保持不变

## 日历与会议室

### 日历视图

- 按月查看日历，支持前后翻月
- 侧栏展示当日和近期日程
- 日程卡片：标题、时间、参与人、会议室信息
- 点击日期或「新建日程」按钮创建日程

### 创建日程

- 填写标题、描述
- 选择开始时间、结束时间
- 添加参与人（搜索选择组织成员）
- 选择会议室（带实时冲突检测）

### 会议室预订

- 创建日程时可选择会议室
- 显示会议室名称、楼栋、楼层、房间号、容纳人数
- **实时冲突检测**：选择的时间段内若该会议室已被占用，前端即时提示并阻止提交
- 日程卡片展示关联的会议室信息

### 会议室管理（管理员）

- 添加 / 编辑 / 删除会议室
- 设置会议室属性：名称、楼栋、楼层、房间号、容量、设施（投影仪/白板/视频设备/电话等）
- 会议室状态：可用 / 维护中 / 停用
- 按楼栋分组展示
- 搜索会议室

## 多维表格（Base）

> 面向结构化协作数据的多维表格，支持多种视图灵活管理业务信息。

### 应用管理

- 创建多维表格应用（名称 + 图标 + 颜色）
- 编辑 / 删除应用
- 应用列表以卡片形式展示

### 数据表

- 每个应用可包含多张数据表
- 支持新建、重命名数据表
- 左侧标签页切换数据表

### 字段系统

支持以下字段类型：

| 字段类型 | 说明 |
|----------|------|
| `text` | 单行文本 |
| `number` | 数字 |
| `select` | 单选（自定义选项 + 颜色） |
| `multi_select` | 多选 |
| `date` | 日期 |
| `checkbox` | 复选框 |
| `url` | 链接 |
| `email` | 邮箱 |
| `phone` | 电话 |
| `rating` | 评分（1-5 星） |
| `currency` | 货币（可设置货币符号） |
| `percent` | 百分比 |
| `attachment` | 附件 |
| `user` | 成员 |

- 首字段自动设为主字段（不可删除）
- 支持添加、编辑、删除字段

### 视图

#### 表格视图（Grid）

- 类似电子表格的行列布局
- 直接在单元格中编辑数据
- 显示行号
- 底部「添加记录」按钮

#### 看板视图（Kanban）

- 按单选字段分组展示记录卡片
- 拖拽风格的列式布局
- 每列显示记录数量
- 「未分类」列收纳无值记录
- 列内直接新增记录

#### 表单视图（Form）

- 以表单方式逐字段填写
- 适合数据收集场景
- 提交后自动创建记录

### 记录管理

- 新增 / 编辑 / 删除记录
- 不同字段类型提供对应的编辑器（文本框、数字输入、下拉选择、日期选择器、星级评分等）

## 企业自建机器人

### 机器人管理（管理员）

- 创建机器人：名称、描述、头像
- 自动生成唯一 API Token
- 配置 Webhook URL 和 Webhook Secret
- 启用 / 停用机器人
- 重新生成 Token
- 删除机器人
- 查看订阅的会话列表

### 机器人开放 API

- **发送消息**：`POST /api/bot/messages` — 向指定会话发送文本/图片/文件消息
- **拉取消息**：`GET /api/bot/messages` — 获取会话历史消息（支持分页）
- **订阅会话**：`POST /api/bot/subscribe` — 订阅会话事件
- **查看订阅**：`GET /api/bot/subscribe` — 查看已订阅的会话
- **取消订阅**：`DELETE /api/bot/subscribe` — 取消订阅

> 详细 API 文档见 [Bot API 文档](./bot-api.md)

### Webhook 回调

- 订阅的会话中有新消息时，自动向机器人 `webhook_url` 推送事件
- 请求头附带 `X-Bot-Secret` 用于验证来源
- 支持自动回复等场景

### 用户直接对话

- 工作台机器人卡片提供「对话」按钮
- 自动创建/复用用户与机器人的专属会话
- 用户发消息即触发 Webhook → 机器人通过 API 回复 → 实时推送至用户

## 工作台

- 个性化问候语（根据时段：上午好/中午好/下午好/晚上好）
- 快捷入口卡片：消息、通讯录、云文档、多维表格、日历、管理后台（仅管理员可见）
- 企业机器人列表：展示所有活跃机器人（名称、描述、状态、创建者、创建时间）
- 一键发起机器人对话

## 企业管理后台

> 仅企业 Owner（管理员）可访问，所有操作记录至审计日志。

### 仪表盘

- 企业概览统计：成员数、会话数、消息数、文档数等

### 成员管理

- 成员列表：姓名、部门、职位、工号、角色、入职时间
- 搜索成员（姓名、邮箱、工号、手机号）
- 按部门筛选
- 成员详情侧栏：
  - 编辑姓名、性别、手机号、部门（下拉选择）、职位、工号、工作城市
  - 修改角色（owner / member）
  - 移除成员
- 邀请成员：
  - 通过邮箱发送邀请（支持批量）
  - 设置邀请有效期（1天/3天/7天/30天/永不过期）
  - 企业邀请码（一键复制）
- 邀请记录标签页：
  - 查看所有邀请的状态（待接受/已接受/已过期/已取消）
  - 复制邀请链接
  - 取消未使用的邀请

### 部门管理

- 树状组织架构展示
- 创建 / 编辑 / 删除部门
- 拖拽调整层级（父部门）
- 查看部门成员数

### 机器人管理

- 机器人 CRUD（创建/查看/编辑/删除）
- Token 管理与重新生成
- Webhook 配置
- 状态切换（启用/停用）

### 会议室管理

- 会议室 CRUD
- 设施配置（投影仪、白板、视频设备、电话等）
- 状态管理（可用/维护中/停用）
- 按楼栋分组

### 加入申请审批

- 查看待处理的加入企业申请
- 批准 / 拒绝申请

### 操作日志

- 记录所有管理员操作（创建/编辑/删除成员、部门、机器人、会议室等）
- 按时间倒序展示
- 显示操作类型、操作对象、详细信息、操作时间

### 企业设置

- 查看/编辑企业名称
- 企业基本信息管理

## 认证与企业体系

### 用户认证

- 邮箱注册（姓名 + 邮箱）
- Cookie 认证（`skylark-uid`）
- 预留 Cloudflare Access / Zero Trust 集成（通过 `Cf-Access-Authenticated-User-Email` Header）
- 登出

### 企业管理

- 创建新企业
- 通过邀请码加入企业
- 通过邀请链接加入企业（管理员邮箱邀请）
- 企业切换（多企业支持）
- 申请加入企业（需管理员审批）

### 邀请系统

- 管理员通过邮箱邀请成员
- 支持批量邀请
- 邀请链接预览页面（展示企业信息、邀请人、有效期）
- 邀请状态：待接受 / 已接受 / 已过期 / 已取消
- 自动过期清理
